# 🎵 Music Recommendation System

This is a Spotify-style Music Recommendation System built using Machine Learning and Streamlit.

## 🚀 Features

- Song recommendation based on similarity
- Clean Spotify-like UI
- Easy to use

📖 Overview

This project recommends songs by analyzing features like artist and genre.
It uses Natural Language Processing (NLP) and similarity metrics to identify and suggest songs that are most similar to the selected track.

📌 Machine Learning Approach

- This project uses a Content-Based Recommendation System
- Recommends songs based on similarity between features
- Does not use traditional supervised learning models

🤖 Algorithms & Techniques Used

- TF-IDF Vectorization
- Converts song data (artist, genre, etc.) into numerical vectors using Natural Language Processing
- Cosine Similarity
- Measures similarity between songs and finds the most relevant recommendations
- Content-Based Filtering
- Suggests songs similar to the user's selected input

✨ Key Features

- 🎧 Personalized song recommendations
- 🎨 Spotify-like modern UI
- ⚡ Fast and interactive web application
- 🧠 Machine Learning-based similarity model
- 📊 Simple and scalable dataset handling

🚀 Deployment
The application is deployed using Streamlit Cloud, enabling direct access via a web browser without any setup.

## 🛠️ Tech Used
- Python
- Pandas
- Scikit-learn
- Streamlit
